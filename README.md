# 🥗 AI Food Nutrition Analyzer

## 📌 Overview  
AI Food Nutrition Analyzer is a Streamlit-based web application that provides detailed nutritional insights for food items. It calculates calories, macronutrients (proteins, carbs, fats), and visualizes results using charts. The app helps users make informed dietary and health decisions through an interactive interface.

## ⚙️ Installation & Setup  

### 🔹 1. Clone Repository  
git clone https://github.com/your-username/ai-food-nutrition-analyzer.git  
cd ai-food-nutrition-analyzer  

### 🔹 2. Create Virtual Environment  
python -m venv .venv  
.venv\Scripts\Activate.ps1  

### 🔹 3. Install Dependencies  
pip install -r requirements.txt  

### 🔹 4. Run the Application  
streamlit run ai-nutrition/app.py  

OR  
python -m streamlit run ai-nutrition/app.py
